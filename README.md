# Idle Wanderer v0.7.0 — Town Crafting

Mobile-first GitHub Pages build.

## New in this version
- Every town now opens the same working Crafting Table menu.
- Crafting recipes check Crafting level and owned materials.
- Craft 1 or craft the maximum possible amount.
- Refine Copper, Iron, Silver, Pyrite, Gold, and Crystal into bars.
- Craft Stone through Crystal axes, pickaxes, daggers, swords, shields, and armour.
- Craft Cedar through Redwood bows.
- Higher-tier axes and pickaxes are automatically used from inventory and reduce gathering time.
- Crafted items include descriptions and equipment/tool statistics.
- Coal remains a gathered crafting/furnace material for later recipes rather than becoming a gear tier.
